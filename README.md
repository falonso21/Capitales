# Quiz de Capitales del Mundo

Aplicacion web interactiva para poner a prueba tus conocimientos sobre las capitales del mundo.

## Modos de juego

- **Adivinar la capital**: Se muestra un pais y debes escribir su capital
- **Adivinar el pais**: Se muestra una capital y debes escribir el pais

## Tipos de partida

- **Modo libre**: Preguntas aleatorias sin limite
- **50 preguntas**: 50 preguntas con estadisticas al final

## Como ejecutar localmente

```bash
pip install streamlit
streamlit run app.py
